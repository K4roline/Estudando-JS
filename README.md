Neste repositório temos alguns projetos utilizados para estudo e treinamento em JS.
São eles:

- Contador de asteriscos
- Encontre o string
- Faixa de preço
- FizzBuzz
- Gerador de endereço
- Maximo entre valores
- Média de nota escolar
- Medidor de velocidade
- Multiplos de 3 e 5
- Numeros primos
- Par ou impar
- Postagem de blog
- Segurança virtual
- Trocando valores
